# BASIC INSTALLATION (CPU only):
# --------------------------------
# pip install -r requirements.txt
#
#
# WITH GPU SUPPORT (NVIDIA CUDA):
# --------------------------------
# 1. Install basic requirements:
#    pip install -r requirements.txt
#
# 2. Install PyTorch with CUDA:
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
# 3. Verify GPU:
#    python -c "import torch; print(f'CUDA Available: {torch.cuda.is_available()}')"
#
#
# UPGRADING PACKAGES:
# -------------------
# pip install -r requirements.txt --upgrade
#
#
# FREEZING CURRENT ENVIRONMENT:
# -----------------------------
# pip freeze > requirements-freeze.txt

# ----------------------------------------------------------------------------
# Troubleshooting
# ----------------------------------------------------------------------------
#
# ERROR: "No matching distribution found"
# SOLUTION: Update pip: python -m pip install --upgrade pip
#
# ERROR: "Microsoft Visual C++ 14.0 is required" (Windows)
# SOLUTION: Install Visual Studio Build Tools
#
# ERROR: "Failed building wheel for scipy"
# SOLUTION: Install system dependencies or use conda
#
# ERROR: CUDA version mismatch
# SOLUTION: Match PyTorch CUDA version with system CUDA version
#
